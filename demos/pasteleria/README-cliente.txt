CATÁLOGO BASE - LABURO DIGITAL / PEDIDO LISTO

Esta carpeta es una plantilla reutilizable para crear catálogos online con pedido por WhatsApp.

ESTRUCTURA

index.html
- Estructura principal de la página.
- No editar salvo cambios grandes de layout.

styles.css
- Diseño visual.
- Editar con cuidado.

app.js
- Funcionamiento del catálogo, filtros, detalle de producto, pedido y WhatsApp.
- No editar para cambiar productos o textos.

config.js
- Datos del negocio.
- Este es uno de los archivos principales para adaptar una marca.

products.js
- Productos, precios, fotos, categorías, ofertas, variantes y disponibilidad.
- Este es el otro archivo principal para adaptar una marca.

robots.txt / sitemap.xml
- Archivos base para publicación y SEO básico.
- Actualizar dominio cuando se publique.

CÓMO ADAPTAR A UN CLIENTE

1. Duplicar la carpeta completa.
2. Cambiar el nombre de la carpeta.
3. Editar config.js.
4. Editar products.js.
5. Abrir index.html y probar.
6. Revisar en modo celular.
7. Probar detalle de producto.
8. Probar pedido por WhatsApp.

REGLAS DE UX

- La tarjeta del producto NO debe tener “Agregar” y “Pedir” al mismo tiempo.
- La tarjeta debe llevar a “Ver detalle”.
- Dentro del detalle se eligen opciones y recién ahí se agrega al pedido.
- El botón de WhatsApp aparece en “Tu pedido”, no todo el tiempo como botón invasivo.
- El mensaje de WhatsApp debe sonar como pedido real, no como consulta genérica.

CHECKLIST ANTES DE MOSTRAR

[ ] Se entiende en 5 segundos.
[ ] Funciona bien en celular.
[ ] Las imágenes no se ven repetitivas.
[ ] El detalle del producto abre bien.
[ ] Se pueden elegir opciones.
[ ] Se puede agregar al pedido.
[ ] El pedido muestra productos y opciones.
[ ] WhatsApp arma un mensaje correcto.
[ ] No hay textos internos de demo.
[ ] No promete pagos, stock o panel admin si no existen.
[ ] El dominio/link de presentación se ve profesional.


ACTUALIZACIÓN UX
- En celular el catálogo usa grilla de dos columnas cuando el ancho lo permite, para evitar una lista interminable de productos.
- Las tarjetas muestran una sola acción: Ver detalle.
- La compra/pedido se confirma desde la ficha ampliada del producto con Agregar al pedido.
- El botón final de WhatsApp queda dentro de Tu pedido, no en cada producto.
