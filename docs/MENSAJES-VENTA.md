# Mensajes de venta

## Primer contacto frío

Hola, vi que venden por Instagram y capaz les serviría tener un catálogo online para que sus clientes vean productos, precios y les manden el pedido directo por WhatsApp.

Estoy armando este tipo de catálogos para negocios chicos. Te puedo mostrar una muestra si te interesa.

## Si responde con interés

La idea es simple: tus clientes entran al catálogo desde Instagram o WhatsApp, ven los productos ordenados, eligen lo que quieren y te llega el pedido armado por WhatsApp.

También se puede hacer editable para que puedas cambiar productos, precios u ofertas sin depender de mí.

## Precio inicial

La versión editable arranca aproximadamente entre USD 200 y 250 para primeros trabajos, según cantidad de productos y nivel de personalización.

Incluye catálogo mobile, pedido por WhatsApp, publicación online, SEO básico y una ronda de ajustes antes de publicar.

## Cierre con seña

Para avanzar trabajo con 50% de seña y el resto al publicar. Antes de publicarlo te muestro una vista previa para revisar textos, productos y funcionamiento.
