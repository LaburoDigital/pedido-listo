# Precios iniciales

Los precios son rangos orientativos para primeros clientes. No son garantía.

## Producto principal: Pedido Listo Editable

Catálogo online editable para negocios que venden por Instagram y WhatsApp.

Precio inicial recomendado: USD 200–250 para primeros casos.
Precio con más portfolio: USD 300–450.

Incluye:

- Catálogo mobile first.
- Productos/categorías.
- Pedido por WhatsApp.
- Ofertas/destacados/disponibilidad.
- Publicación online.
- SEO básico.
- Guía para que el dueño edite productos cuando esté conectada la fuente editable.

No incluye:

- Mantenimiento mensual obligatorio.
- Cambios manuales infinitos.
- Atención a clientes.
- Manejo de redes.
- Pagos online.
- Stock avanzado.
- Panel admin real.

## Entrada excepcional: catálogo básico

Precio: USD 120–180.

Usar solo si el cliente tiene pocos cambios o presupuesto bajo.

## Pro / mini sistema

Precio: USD 450+.

Solo si hay cliente real y necesidad clara.

Puede incluir reservas, panel, base de datos, pedidos avanzados o integraciones.

## Pagos

Recomendado:

- 50% de seña antes de producción final.
- 50% al publicar.

No publicar versión final ni conectar dominio sin seña.
