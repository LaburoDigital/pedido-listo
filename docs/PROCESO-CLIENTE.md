# Proceso de cliente

## 1. Lead

Detectar negocio con problema claro:

- vende por Instagram/WhatsApp;
- no tiene catálogo ordenado;
- recibe muchas preguntas repetidas;
- productos/precios dispersos;
- imagen comercial mejorable.

## 2. Contacto

Enviar mensaje corto y específico. No vender “webs”; vender orden y pedidos por WhatsApp.

## 3. Demo o preview

Mostrar demo genérica o preview ligera. No entregar una web final gratis.

## 4. Seña

Si quiere avanzar, pedir 50% antes de producción final.

## 5. Materiales

Pedir:

- nombre de marca;
- logo si tiene;
- WhatsApp;
- Instagram;
- productos;
- precios;
- fotos;
- condiciones de retiro/envío;
- colores/referencias.

## 6. Producción

Duplicar plantilla. Adaptar `config.js` y `products.js`. Tocar `app.js` solo si el rubro lo exige.

## 7. QA

Pasar `QA-CHECKLIST.md` antes de mostrar.

## 8. Revisión

Una ronda de cambios incluida.

## 9. Publicación

Publicar con link profesional. Evitar subdominios gratuitos como cara final si el cliente lo va a usar públicamente.

## 10. Saldo

Cobrar saldo al publicar.

## 11. Cambios posteriores

No hay mantenimiento obligatorio. Cambios puntuales se cotizan aparte. Si el cliente cambia seguido, recomendar versión editable.
