# Documentación de Pedido Listo

Esta carpeta contiene el método de trabajo del producto.

Archivos:

- `CHANGELOG.md`: historial de cambios por versión.
- `QA-CHECKLIST.md`: pruebas obligatorias antes de mostrar una demo.
- `PRECIOS.md`: paquetes y límites comerciales.
- `MENSAJES-VENTA.md`: mensajes base para contactar clientes.
- `PROCESO-CLIENTE.md`: proceso desde lead hasta entrega.
